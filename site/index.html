不是聊天壳，而是学习对象之间的连接。

教材范围提供上下文，习题与错题形成长期记录，概念记忆和复习队列连接下一次行动。

教材问答

在选定教材范围内检索、生成并保留来源。

问答链组合知识图谱精确命中、向量检索、词法索引、语义角色与相邻片段。SSE 按阶段返回，单个检索源异常时可以降级。

习题工作区

从题目导入到练习会话。

保留来源、章节、题型、难度、概念、状态与练习历史，支持批量导入和回滚。

错题复习

记录错因，并按到期队列复习。

手动录入、筛选、复习历史和 SM-2 兼容调度已形成闭环。OCR 结果必须人工校正。

一次问答如何变成可继续的学习记录。

每一步对应仓库中已经存在的数据对象和调用链。模型生成不是唯一入口，本地错题和习题功能可以独立使用。

  1. 导入与整理

    导入教材或已有 MinerU 结果，建立章节、片段与索引。

  2. 范围内问答

    在学科和教材范围内规划意图，执行混合检索并流式生成。

  3. 练习与归因

    将题目保存在习题库，把错误答案、错因和来源沉淀到错题本。

  4. 复习与回看

    按到期队列复习，在学习情况中回看概念接触和薄弱信号。

桌面壳、本地 API 与学习数据分层。

Electron 负责桌面生命周期和用户数据目录；React 提供交互；FastAPI 编排教材、练习与复习服务；不同存储保存不同类型的数据。
