# Project layout

This project is a local postgraduate exam study assistant. The repository contains application source code, deployment files, tests, and local-only runtime data.

## Application source

```text
backend/      FastAPI API routes and schemas
frontend/     React + Vite web UI
agents/       legacy/utility agent wrappers
graph/        LangGraph workflow, planner, retrieval, generation, feedback
ingestion/    PDF parsing, OCR adapters, textbook ingestion, vector store
knowledge/    knowledge graph, concept memory, keyword index, visualization
memory/       mistake book, exercise bank, study memory, spaced repetition
utils/        shared text/LaTeX/thinking-filter helpers
ui/           CLI and legacy Gradio UI
```

## Deployment and operations

```text
Dockerfile                     one-container full-stack image
docker-compose.yml             local Docker deployment
.dockerignore                  excludes local data/secrets from image build
.env.example                   safe config template
scripts/docker-entrypoint.sh   container startup helper
scripts/update-docker.ps1      backup + build/pull + restart + health check
scripts/backup-docker-data.ps1 host data backup helper
scripts/export-new-user-package.ps1 clean source export for a new user
```

## Documentation and tests

## Root-level scratch files

Files such as `_test_*.py`, `_benchmark*.py`, `_output.txt`, `ocr_*.json`, and `session_*.zip` are local diagnostics or temporary artifacts. They are ignored and should not be part of a clean deployment.

The current clean export path is generated by:

```powershell
.\scripts\export-new-user-package.ps1
```
